angular2-demo

Prerequisites

  • Install Node.js

    • on OSX use homebrew brew install node

    • on Windows use chocolatey choco install nodejs

  • Clone pokedex's repo

    git clone https://github.com/yechuzheng/angular2-demo.git

  • Install npm dependencies

    npm install

  • Start and enjoy

    npm start
